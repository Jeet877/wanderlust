A full-stack travel listing web application that allows users to discover, create, review, and manage travel destinations and stays.

---

## 🚀 Features

- 🔐 User Authentication & Authorization
- 🏡 Create, Edit & Delete Listings
- 📸 Image Uploads with Cloudinary
- ⭐ Reviews & Ratings System
- 🛡️ Secure Route Protection
- ✅ Server-side Validation
- 💬 Flash Messages & Error Handling
- 📱 Responsive Design
- 👤 User-specific Listing Management

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Jeet877/wanderlust.git
```

### Navigate to Project

```bash
cd wanderlust
```

### Install Dependencies

```bash
npm install
```

### Create .env File

```env
ATLASDB_URL=your_mongodb_connection_string

SECRET=your_secret_key

CLOUD_NAME=your_cloudinary_name
CLOUD_API_KEY=your_cloudinary_api_key
CLOUD_API_SECRET=your_cloudinary_api_secret
```

### Run Project

```bash
node app.js
```

or

```bash
npm start
```
